# 경대북문지기 매출 분석 시스템

Tkinter 기반 GUI를 활용하여  
경북대학교 북문 주변 상권의 매출 데이터를 분석하는 프로그램입니다.

---

## 주요 기능
- CSV 파일(euc-kr) 매출 데이터 불러오기
- 총 매출 기준 상위 N개 가맹점 분석
- 방학 / 비방학 시즌 매출 비교 분석
- Matplotlib, Seaborn 기반 그래프 시각화
- 분석 결과 PNG / PDF 저장

---

## 실행 환경
- OS: Windows
- Python: 3.10 이상
- GUI Framework: Tkinter

---

## 실행 방법

```bash
pip install -r requirements.txt
python main.py
